# Requirement: Mindful Budget Planning

**Actor:** End users
**Domain:** Product

## Purpose

### Driver
People set financial goals but have no structured way to translate them into daily spending limits. They start each month with good intentions but lose track within days because the gap between "save 200€ this month" and "can I buy this coffee?" is too wide.

### Requirement
**End users** need a simple monthly budget planner that calculates a daily spending allowance so that they can make confident daily spending decisions without mental math.

## Priority
High

## Type
Operations

## Notes
This is the core Kakeibo flow: set income, set savings goal, allocate to four categories, get a daily budget. Currently partially implemented (budget creation + daily view exist).
