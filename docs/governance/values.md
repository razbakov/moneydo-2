# MoneyDo — Values

## Adopted S3 Principles

- **Effectiveness** — Devote time only to what brings you closer to achieving objectives
- **Consent** — Raise, seek out, and resolve objections to proposals and policies
- **Empiricism** — Test all assumptions through experiments and continuous revision
- **Continuous Improvement** — Regularly review outcomes, then make incremental improvements
- **Equivalence** — Involve people in making and evolving decisions that affect them
- **Transparency** — Record all valuable information and make it accessible to everyone
- **Accountability** — Respond when something is needed, do what you agreed to do

## Additional Values

- **Simplicity** — Resist feature creep. Every addition must justify its complexity. When in doubt, leave it out.
- **Mindfulness** — The product philosophy extends to how we build it. Intentional decisions over reactive ones.
- **Privacy first** — User financial data is sacred. No ads, no data selling, no analytics that compromise privacy.
